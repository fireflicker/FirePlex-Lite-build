# GitHub Build Steps

1. Create a new empty GitHub repository.
2. Upload all files/folders from this project, not the zip.
3. Make sure these exist:
   - app/
   - .github/workflows/build-apk.yml
   - build.gradle.kts
   - settings.gradle.kts
   - gradle.properties
4. Commit changes.
5. Open Actions.
6. Run Build APK.
7. Download artifact FirePlex2Clean-debug-apk.
8. Install app-debug.apk.
